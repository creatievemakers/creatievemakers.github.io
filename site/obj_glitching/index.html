  • 1. what is an obj file?

      • vertex (id, position)
      face (corresponding vertecis)
      texture (u & v coordinates between 0-1)

  • download an obj file.

      • sketchfab cc0 collection

      starting geometry

  • import into blender and sort the points according to the proximity to the 3d-cursor

      • open blender and delete everything (a + x + enter)
      import model
      enable statistics in the overlays tab. if the vertecis are less than 10 000, add a subdivision modifier and apply it.
      go into "edit mode", (optional: delete unwanted parts with face selectionb) and select a point of interest.
      set your 3d cursor to that points (shift + s -> cursor to selected)
      select all geometry and got to mesh -> sort elements -> by cursor distance.
      export geometry as obj_sorted.obj file. make sure to disable "write normals" and "write materials".
      rename the obj_sorted.obj file to obj_sorted.txt.

  • prepare the file in excel

      • open the .txt file in excel and seperate the data according to spaces. (ctrl+a -> data -> 'Text to columns' -> delimited by spaces)
